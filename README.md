# Feedback Systeem For Unreal
Developed with Unreal Engine 5.6.1

# What This Plugin Does
This plugin provides an integrated feedback system directly inside Unreal Engine, allowing developers and designers to give and receive feedback without leaving the editor.

# Key Features
Add feedback directly in the editor while working Keep feedback tied to specific contexts (scenes, assets) Reduce workflow interruptions by eliminating external tools Provide clearer communication between team members Help track what needs improvement and what actions are required

# Installation Guide
Follow these steps to install the plugin into your Unreal Engine project:

Download the Plugin Clone the repository or download it as a ZIP from GitHub Extract the files if you downloaded a ZIP

Locate Your Project Folder Open your Unreal project directory

Find (or create) a folder named: "Plugins"

Place the plugin folder inside: YourProject/Plugins/

Open the Project Launch your project in Unreal Engine

Enable the Plugin Go to: Edit -> Plugins Search for "feedbacksystem" Enable it and restart the editor if prompted
